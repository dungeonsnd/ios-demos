# ios-demos
ios demos

本项目包含了一些简单示例，以及一些常用的c/c++开源库的ios移植，方便其他开发者。
由于笔者提供了一些编译好的开源库ios移植静态库，所以体积会非常大。如果你git clone本项目，请视网络速度耐心等待。


很多网友提供的编译开源库的脚本中没有启用BITCOD.
选项 -fembed-bitcode 是用来启用bitcode的。如果不启用，那么使用者(新的xCode默认启用BITCODE选项)将无法编译通过，除非使用者把他自己项目中的全局bitcode禁用，而这对使用者非常不友好。
笔者在本项目中提供的移植将尽量提供bitcode版本的编译方式，方便开发者使用。
